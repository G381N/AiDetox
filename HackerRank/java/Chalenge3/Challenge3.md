Title: Weird or Not Weird

Task: Given integer N, print "Weird" or "Not Weird" using rules:
- If N is odd -> "Weird"
- If N is even and 2<=N<=5 -> "Not Weird"
- If N is even and 6<=N<=20 -> "Weird"
- If N is even and N>20 -> "Not Weird"

Input: One integer N

Output: Print either "Weird" or "Not Weird"

Examples:
- Input: 3 -> Output: Weird
- Input: 24 -> Output: Not Weird
